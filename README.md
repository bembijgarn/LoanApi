## Features

* User registration;
<pre>{
  "FirstName": "string",
  "LastName": "string",
  "Age": 0,
  "Email" : "Email",
  "SalaryPerMont" : 0,
  "UserName": "string",
  "Password": "string",
}</pre>
* User Login;
<pre>{
  "UserName" : "String",
  "Password" : "String"
}</pre>
* Password hashing;
* Role-based authorization;
* User Login via access token creation;
 * Role - Accountant;
 * Generate Role Accoutant ;
 * Accoutant UserName/Passowrd;
<pre>
{
  "userName": "admin000",
  "password": "admin007"
}</pre>
 


* Role - Accountant
  * Get All User
  * Delete Loan Of User
  * Delete User By User Id
  * Accept Loan By User Id
  * Block Loan By User Id
  * Block/Unblock"
  <pre>{
      "Id" : 0,
      "Block" : "Block/Unblock"
  }</pre>
  * Update Loan By User Id
  * Update User Status
  * Get Loan By User Email
  
* Role - User
  * Add Loan (if IsBlocked = false)
  * Delete Loan (if IsBlocked = false, Loan Status = "Requested")
  * Update Loan (if IsBlocked = false, Loan Status = "Requested")
  * Update User Info (if IsBlocked = false)
  * Get User Info
  * Get Loan 
  * Change Password
